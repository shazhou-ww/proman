---
scenario: "cards query --sources returns cards associated with given source files"
feature: cards
tags: [cards, query, sources]
---

## Given
- `.cards-index.json` exists (previously generated by `proman cards index`)
- Index contains:
  - `src/plugins/loader.ts` → `["nerve-plugin-system"]`
  - `src/plugins/registry.ts` → `["nerve-plugin-system"]`
  - `src/config/loader.ts` → `["config-loading"]`

## When
- Run `proman cards query --sources src/plugins/loader.ts src/config/loader.ts`

## Then
- Reads `.cards-index.json` and looks up each file in `by_source`
- Outputs matching card IDs (deduplicated):
  ```
  nerve-plugin-system
  config-loading
  ```
- Exit code 0
- If a source file has no associated cards, it is silently skipped
- If `.cards-index.json` does not exist, exits with error and message suggesting to run `proman cards index` first
